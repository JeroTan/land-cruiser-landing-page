## Why

Toyota Mobility Solutions Philippines needs a high-impact, premium landing page to showcase the Land Cruiser. By combining Apple's cinematic, scroll-driven storytelling methodology with the official Toyota Motor Corporation Brand Design system, the page will deliver a visually stunning, immersive experience that highlights the vehicle's features while maintaining strict adherence to Toyota's global brand identity.

## What Changes

- Create a new single-page landing page for the Land Cruiser.
- Implement Apple-style layout architecture: full-bleed hero, sticky quick-highlights navigation, alternating feature introductions, stat rows, and scroll-triggered animations.
- Integrate the official Toyota Motor Corporation Brand Design System: Toyota Type typography (or Inter fallback), Toyota brand colors (Toyota Red, Black, White, Gray), and the 4px base spacing grid.
- Add scroll-triggered entrance animations, count-up stats, and parallax effects following the Apple landing page methodology.

## Capabilities

### New Capabilities
- `landing-page-ui`: Core UI layout, navigation, and content sections for the Land Cruiser landing page.
- `scroll-animations`: Implementation of IntersectionObserver-based scroll animations, count-ups, and parallax effects.
- `brand-theme`: Integration of official Toyota Motor Corporation Brand Design tokens, colors, typography, and spacing within the Apple-style component structure.

## Impact

- Adds a new route/page for the Land Cruiser campaign.
- Introduces new reusable UI components aligned with the Apple methodology (Hero, Sticky Nav, Feature Grids).
- Requires global configuration for Toyota Type fonts, Toyota brand colors, and the 4px spacing system in the project's styling setup.
